# Глава 8. ФИКСАЦИЯ БИЗНЕС-РЕЗУЛЬТАТА (PRESENT PERFECT)

## 1. Суть
Present Perfect (have/has + V3) используется для фиксации готового результата к текущему моменту. Время совершения действия неважно, в отличие от Past Simple.
*   *I have written the script.* (Скрипт готов, результат на руках).

## 2. Утверждение
`[Субъект] + [have / has] + [Глагол V3]` (для He/She/It — `has`).
*   **Правильные:** -ed (*checked*).
*   **Неправильные:** 3-я колонка (*written*).
*   *Пример:* The system has processed the timesheet.

## 3. Отрицание и вопрос
`HAVE/HAS` работают самостоятельно, `do/does/did` не нужны.
*   **Отрицание:** `haven't` / `hasn't`. *The contractor hasn't submitted the documentation yet.*
*   **Вопрос:** Инверсия (`Have`/`Has` в начало). *Have you cleaned the database?*

## 4. Ограничения (Time Markers)
**Категорически нельзя** использовать с точными метками времени (yesterday, last week). Метка времени требует Past Simple.
*   *Ошибочно:* I have written the code yesterday.
*   *Правильно:* I wrote the code yesterday.

## 5. Примеры
*   (+) We have integrated the database.
*   (-) I haven't deployed the script yet.
*   (?) Has the inspector approved the journal?
