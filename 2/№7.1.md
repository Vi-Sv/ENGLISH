# Глава 7. ЛОГИ ПРЯМОГО ЭФИРА (PRESENT CONTINUOUS)

## 1. Суть: Процесс в реальном времени
Present Continuous фиксирует действие, происходящее непосредственно в момент речи [1].
*   **Сравнение:** *I write* (вообще) vs *I am writing* (сейчас).

## 2. Утверждение: Гибридный движок
Конструкция: `[Субъект] + [am / is / are] + [Глагол + ING]` [1].
*   *The server **is uploading** the database.*

## 3. Отрицание и Вопрос
Сильный оператор `TO BE` работает без помощников.
*   **Отрицание:** *I **am not reviewing** the code.*
*   **Вопрос:** ***Are** you watching the logs?*

## 4. Фильтр: Глаголы состояния (STATIVE VERBS)
Глаголы, обозначающие внутреннее состояние (знание, чувства, владение), **не используются** в `-ING` форме [1].
*   *Неправильно:* I am understanding...
*   *Правильно:* *I **understand** the code now.*

## 5. Примеры
*   *We **are configuring** the server.*
*   *The script **is not running**.*
*   ***Is** the database calculating?*
